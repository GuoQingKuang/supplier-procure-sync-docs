# 数据库设计

## 表结构概览

| 表名 | 说明 | 数据源 |
|------|------|--------|
| delivery_notice | 送料通知单 | MySQL |
| pre_delivery_notice | 预发货单 | MySQL |
| supplier_account | 供应商账号 | MySQL |
| 金蝶采购订单表 | | SQL Server (ERP) |
| 金蝶收料单表 | | SQL Server (ERP) |

## delivery_notice（送料通知单）

| 字段 | 类型 | 说明 |
|------|------|------|
| id | BIGINT | 主键 |
| notice_no | VARCHAR | 通知单号 |
| supplier_id | BIGINT | 供应商ID |
| supplier_code | VARCHAR | 供应商编码 |
| supplier_name | VARCHAR | 供应商名称 |
| purchase_org_id | BIGINT | 采购组织ID |
| purchase_org_name | VARCHAR | 采购组织名称 |
| status | INT | 状态：1-发货中 2-已发货 3-已作废 4-审核中 |
| ship_time | DATETIME | 发货时间 |
| material_data | TEXT | 物料明细JSON |
| print_data | TEXT | 库位/备注JSON |
| kingdee_receive_bill_id | BIGINT | 金蝶收料单内码 |
| kingdee_receive_bill_no | VARCHAR | 金蝶收料单编号 |
| kingdee_update_params | TEXT | 金蝶Save参数JSON |
| fail_reason | TEXT | 失败原因 |
| create_time | DATETIME | 创建时间 |
| update_time | DATETIME | 更新时间 |
| create_by | BIGINT | 创建人 |

## pre_delivery_notice（预发货单）

| 字段 | 类型 | 说明 |
|------|------|------|
| id | BIGINT | 主键 |
| notice_no | VARCHAR | 预发货单号 |
| formal_notice_no | VARCHAR | 关联正式通知单号 |
| status | INT | 状态：草稿/已确认/已提交 |
| warehouse_group | VARCHAR | 库房分组 |
| pending_area | VARCHAR | 待检区仓位 |
| remark | VARCHAR | 备注 |
| ... | | 其他字段 |
