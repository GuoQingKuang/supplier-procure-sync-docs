## 多数据源设计

### 数据源说明
- MySQL：本地业务库（供应商、通知单、预发货单等）
- SQL Server：金蝶 ERP 库（采购订单、收料单等）

### 事务隔离方案

```java
// 外层：MySQL 事务
@Transactional
public void submit() {
    // 本地 MySQL 操作
    deliveryNoticeDao.save(notice);
    
    // 通过 self 代理调用，触发 @DS 切换
    List<EntryRemainQtyVo> result = self.queryErpRemainQty(entryIds);
}

// 内层：SQL Server 独立事务
@DS("erp")
@Transactional(propagation = Propagation.REQUIRES_NEW, ...)
public List<EntryRemainQtyVo> queryErpRemainQty(List<Long> entryIds) {
    return erpMapper.selectBatchRemainQty(entryIds);
}
```
